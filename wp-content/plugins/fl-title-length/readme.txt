Plugin Name: Post Title Length
Description: Restricts the post title length to a predefined limit. When creating or editing posts, users will not be able to enter titles longer than the limit. This plugin is based on the "Limit a post title to X characters" plugin authored by Jean-Philippe Murray:  http://wordpress.org/extend/plugins/limit-a-post-title-to-x-characters/. 
Author: Fluid Studios Team
Author URI: http://studios.fluidproject.org/
Tested Wordpress version: 3.2.1

== Installation ==

The steps is within the context of Fluid Studios

1. Login as admin
2. Dashboard -> Plugins -> Activate "Post Title Length"

== How to use ==

1. Login as admin
2. Go to Dashboard -> Settings -> Post Title Length, adjust the length limit and save
3. Now title lengths will be limited to the value entered both when creating and when editing a post
